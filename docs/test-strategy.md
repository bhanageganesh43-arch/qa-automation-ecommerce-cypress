# Test Strategy

## Scope
- UI automation for core e-commerce journeys on SauceDemo
- API automation for Fake Store API endpoints
- Smoke and Regression suite split

## Objectives
- Validate critical business flow quickly (Smoke)
- Ensure broader feature confidence (Regression)
- Maintain readable, reusable, scalable test design

## Tools
- Cypress + JavaScript
- Mochawesome reporting
- GitHub Actions (CI)

## Test Types
- UI: login, inventory, cart, checkout, logout
- API: products, categories, carts, users
